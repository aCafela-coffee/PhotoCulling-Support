# PhotoCulling

PhotoCulling helps you review photos and conveniently move them to specific folders.  
PhotoCulling은 사진을 검토한 뒤, 신속하게 폴더별로 분류 및 이동할 수 있는 앱입니다.


## How to Use (사용 방법)

### Navigation (이동)
You can move the cursor or navigate through photos using:
* **Arrow Keys** (방향키)
* **WASD** Keys
* **JK** Keys

## Privacy Policy (개인정보 처리방침)

**Effective Date:** 2025-11-23

**1. Data Collection** PhotoCulling does **not** collect, store, or share any personal information. All photo processing and file management operations are performed locally on your device.  
(PhotoCulling은 어떠한 개인정보도 수집, 저장 또는 공유하지 않습니다. 모든 사진 처리 및 파일 관리 작업은 사용자의 기기 내에서 로컬로 수행됩니다.)

**2. File Access** The app only accesses the files and folders you explicitly select to perform the sorting and organizing functions.  
(이 앱은 정렬 및 정리 기능을 수행하기 위해 사용자가 명시적으로 선택한 파일 및 폴더에만 접근합니다.)
