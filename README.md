# PhotoCulling

PhotoCulling helps you review photos and conveniently move them to specific folders.  
PhotoCulling은 사진을 검토한 뒤, 신속하게 폴더별로 분류 및 이동할 수 있는 앱입니다.


## How to Use (사용 방법)

### Navigation (이동)
You can move the cursor or navigate through photos using:
* **Arrow Keys** (방향키)
* **WASD** Keys
* **JK** Keys
